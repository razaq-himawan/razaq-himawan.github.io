## Project Overview

Architected a monolithic web application using Laravel and Inertia.js, implementing Server-Side Rendering (SSR) to maximize performance while maintaining a modern React-based UI.

## Key Contributions

- **Database Optimization:** Engineered a normalized MySQL database schema reducing data redundancy by ~30%.
- **SSR Implementation:** Utilized Inertia.js to bridge Laravel backend with React frontend without building a separate API.
- **Delivery:** Delivered high-priority functional deliverables 2 weeks ahead of schedule.
